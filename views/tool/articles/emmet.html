Dreamweaver版的Emmet插件，除了没有<img>自动填充尺寸功能，也没有图片自动转base64功能，其他都有。

快捷键将是一个考验记性的大麻烦。
Dw对DOM选择功能比较简单，但是Emmet有大量选择功能，如：
前后节点Point选择 (Ctrl+Alt+[ || Ctrl+Alt+])、
前后标签Tag选择 (Ctrl+Alt+E || Ctrl+Alt+D)、
前后单词Item选择 (Ctrl+[ || Ctrl+])，
还有跳到闭合标签 (Ctrl+T)、删除标签 (Ctrl+R)*、合并多行 (Ctrl+Alt+G)、包裹内容 (Ctrl+W)*、添加注释 (Ctrl+/) 等等。

(快捷键) 为本人自定义非默认，*特别有价值。
总体而言，如果熟练掌握了以上快捷键和语法，在HTML编写速度上，提升不是一点点。


Emmet的CSS部分，是以CSS语句的缩写形式，进行快速转义。如： 
m10 5 → margin:10px 5px;
tdu → text-decoration:underline;
lh20 → line-height: 20px;
db → display:block;
fl → float:left;
其理念和我的CSS通用库类似...以单词字母开头为缩写，当然有个别缩写形式存在冲突，可能得不到你想要的转义。

特殊缩写： 
p → %
e → em
x → ex
#1 → #111111
#e0 → #e0e0e0
#fc0 → #ffcc00
p!+m10e!  →  padding:  !important;  margin: 10em !important;
m-10--20 → margin:-10px -20px
总体而言，因为有CSS通用库的存在，Emmet的CSS转义这块实际提升速度没有HTML转义那么明显。


2. 在 Dreamweaver CS6 中安装 emmet

下载点击这里：http://emmet.io/download/ 或者直接 Emmet.zxp

如果已经安装了Adobe的Extension Manager(扩展管理器)，直接双击安装就可以啦。如果没有，先下载扩展管理器。

 

3. 在安装时，如果出现 “无法更新菜单,将不会安装该扩展” 的解决方法

开始->运行->输入“%appdata%/adobe”
删除 ..\Adobe\Dreamweaver CS6\zh_CN\Configuration\Menus 目录下的 Menus.xbk 或 Menus.xml，然后重新启动Dreamweaver即可成功安装扩展，此方法同样适用无法安装 Zen Coding 扩展的朋友。

安装成功了。


4. 默认快捷键 ctrl+e 和 插入标签项 的快捷键冲突，需要修改。

在 Dreameaver，选择“编辑”->"快捷键"命令，打开“菜单命令”->“插入”->“标签”。重新快捷键，比如“ctrl+shift+e”，就避免了和 Emmet 快捷键冲突。
